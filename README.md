### Set up

Initialize repo:

```cli
git clone https://github.com/sagemodeninja/fluent-navigation-view-component.git
cd fluent-navigation-view-component
```

Install the necessary dependencies:

```
npm install
```

Build the project:

```
npm run build
```

Run a local demo:

```
npm start
```
